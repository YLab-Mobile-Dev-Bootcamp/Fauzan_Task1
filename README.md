# Aplikasi Counter

Aplikasi Counter .
Berisi halaman utama dan halaman hasil.
Halaman utama untuk menambah dan mengurangi,
halaman hasil menampilkan hasil yang merupakan
jumlah counter saat ini. Digunakan GetX untuk
state manager dan navigasi.
